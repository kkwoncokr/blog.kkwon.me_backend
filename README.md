# blog.kkwon.me_backend

## kkwon blog's Back-End Source.

#### Git Clone.

```
git clone https://github.com/kkwoncokr/blog.kkwon.me_backend
```

#### Config
```
cd ${project Root}
sudo vim .env
```
### .env Config
```
PORT=OOO
MONGO_URI=mongodb://
JWT_SECRET=OOO
```

#### Node Module Install.
```
npm i yarn
yarn install
```

### Local Develper

```
yarn start
```
